# ResumeIQ AI 🧠

> **Analyze. Edit. Download. Land More Interviews.**

Built by **Sudharsun**

---

## What is ResumeIQ AI?

ResumeIQ AI is a fully browser-based resume tool — no backend, no login, no installation. It does two things in one single HTML file:

1. **Analyze** your existing PDF resume with AI
2. **Build & download** a new ATS-optimized resume from scratch

---

## Features

### 🔍 Resume Analyzer
Upload your PDF resume and instantly get:

| Feature | Description |
|---|---|
| ATS Score | Animated score ring — color-coded pass / borderline / fail |
| Missing Keywords | Shows exactly what's absent that recruiters look for |
| Skill Extraction | Detects 35+ technical skills and 15+ soft skills |
| AI Suggestions | Personalized tips to strengthen every section |
| Live Dashboard | Populates with your real resume data after upload |
| Community Reviews | Submit star ratings and written feedback |

### ✏️ Resume Editor (built-in)
Click **"Edit Resume"** in the nav to open a full-screen editor overlay:

| Feature | Description |
|---|---|
| Live Preview | See your resume update in real time as you type |
| ATS-Safe Format | Black & white, Times New Roman — no colors, no icons |
| Sections | Personal info, summary, experience, education, skills, certs, projects, awards, languages |
| Smart Inputs | "Currently working here" toggle, comma-separated skill entry, quick-add skill chips |
| Completeness Bar | Tracks how complete your resume is with next-step hints |
| PDF Download | One-click export to a clean A4 PDF named after you |
| Load Sample | Pre-fills a full example resume to get started fast |
| 100% Private | All data stays in your browser — nothing is uploaded |

### 🎨 UI & Experience
- Floating particles background
- Cursor glow effect
- 3D card tilt on hover
- Scroll-reveal animations
- Mobile responsive
- Dark glass UI

## How to Use

### Option 1 — Live site
Visit the GitHub Pages URL above. No install needed.

### Option 2 — Run locally
Download `index.html` and open it in any modern browser. That's it — one file, zero dependencies to install.

---

## Tech Stack

| Layer | Technology |
|---|---|
| UI Framework | React 18 (CDN UMD) |
| JSX | Babel Standalone |
| Styling | Tailwind CSS v3 (Play CDN) + custom CSS |
| PDF Reading | PDF.js 3.11 (client-side) |
| PDF Export | jsPDF 2.5 + html2canvas 1.4 |
| Animations | CSS keyframes + Intersection Observer |
| Build | None — single `index.html` file |

---

## Privacy

Everything runs **entirely in your browser**. PDF text extraction uses PDF.js locally. Resume data never leaves your device. No analytics, no tracking, no servers.
